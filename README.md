# Final-year-project---G8
Real time health data transmission in a wearable device via 5G
The rapid advancement in healthcare technology has created a growing need for continuous and real-time monitoring of vital human parameters. Traditional health monitoring systems are often limited by delayed data transmission, bulky infrastructure, and lack of accessibility in remote areas. To overcome these limitations, this project presents the design and development of a Real-Time Health Data Transmission System using a Wearable Device integrated with 5G communication technology. The proposed system focuses on developing a compact, efficient, and cost-effective wearable device capable of continuously monitoring essential physiological parameters such as heart rate, blood oxygen level (SpO₂), and body temperature. These parameters are captured using integrated biosensors and processed through a microcontroller-based embedded system. The processed data is then transmitted in real time using advanced wireless communication, primarily leveraging the capabilities of 5G technology, which provides ultra-low latency, high data rates, and reliable connectivity. One of the key features of this system is its ability to provide instant access to health data through cloud integration. The transmitted data is stored and analyzed on a cloud platform, allowing users, caregivers, or medical professionals to monitor health conditions remotely through a mobile or web-based dashboard. This enables early detection of abnormalities and ensures timely medical intervention, which is critical in emergency situations.
